La app se puede ejecutar con python app.py
El laboratorio.html contiene una vista de prueba de concepto, mientras que laboratorio_malicioso.html contiene una versión mejorada donde la captura se realiza en segundo plano.
Las capturas se almacenan en el directorio capturas.
Se puede hacer tunneling a través de ngrok para obtener una url accesible desde cualquier lugar con ngrok http 80
